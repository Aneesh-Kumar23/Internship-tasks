# Data Science & Analytics Internship Tasks

## Overview

This repository contains the tasks completed during my Data Science & Analytics Internship. The projects focus on data analysis, data visualization, machine learning, predictive modeling, and model evaluation using real-world datasets.


# Task 1: Iris Dataset Analysis

## Objective

To explore and visualize the Iris dataset and understand the relationships between different flower characteristics.

## Approach

* Loaded the Iris dataset using Pandas and Seaborn.
* Inspected the dataset using shape, columns, head(), and statistical summaries.
* Checked for missing values and duplicate records.
* Performed Exploratory Data Analysis (EDA).
* Created visualizations including:

  * Scatter Plot
  * Histogram
  * Box Plot
* Analyzed relationships among sepal and petal measurements.

## Results and Insights

* Setosa species was clearly separated from the other species.
* Petal length and petal width showed strong correlation.
* The dataset contained very few outliers.
* Visualizations helped identify patterns and differences among species.


# Task 2: Credit Risk Prediction

## Objective

To predict whether a loan applicant is likely to default on a loan using machine learning classification techniques.

## Dataset

Loan Prediction Dataset

## Approach

* Loaded and explored the loan prediction dataset.
* Identified and handled missing values using appropriate preprocessing techniques.
* Performed data cleaning and feature preparation.
* Visualized important features such as:

  * Loan Amount
  * Applicant Income
  * Education
  * Credit History
* Converted categorical variables into numerical form where required.
* Split the dataset into training and testing sets.
* Trained a Logistic Regression classification model.
* Evaluated model performance using:

  * Accuracy Score
  * Confusion Matrix

## Results and Insights

* Missing value handling improved overall data quality.
* Credit history was one of the most influential features in loan prediction.
* Applicant income and loan amount significantly affected loan approval outcomes.
* The Logistic Regression model achieved reliable classification performance.
* The confusion matrix provided insights into correctly and incorrectly classified cases.


# Task 3: Predicting Insurance Claim Amounts

## Objective

To estimate medical insurance charges based on personal and health-related information.

## Dataset

Medical Cost Personal Dataset

## Approach

* Loaded and explored the insurance dataset.
* Checked for missing values and data consistency.
* Performed Exploratory Data Analysis (EDA).
* Visualized relationships between:

  * Age and Charges
  * BMI and Charges
  * Smoking Status and Charges
* Encoded categorical variables where necessary.
* Split the dataset into training and testing sets.
* Trained a Linear Regression model to predict insurance charges.
* Evaluated model performance using:

  * Mean Absolute Error (MAE)
  * Root Mean Squared Error (RMSE)

## Results and Insights

* Smoking status had the strongest impact on insurance charges.
* Older individuals generally incurred higher insurance costs.
* Higher BMI values were associated with increased medical expenses.
* The Linear Regression model successfully captured the relationship between personal attributes and insurance charges.
* MAE and RMSE metrics were used to measure prediction accuracy and model performance.
